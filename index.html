freedom.js is loading

freedom.js is a framework for building peer-to-peer (P2P) web apps. freedom.js makes it easy to quickly create social interactive applications that instantly work in modern web browsers, Chrome packaged apps, Firefox extensions (coming soon), node.js, and native mobile apps (coming soon). Because freedom.js apps are just JavaScript, they can be distributed as packages on an app store or hosted on static web servers. We're bringing peer-to-peer back, baby.

P2P is reborn in the Web

Build powerful P2P apps using only JavaScript and HTML in a single-page application. Write applications in terms of peers, not IP addresses. Our API lets you subscribe to a peer's online status and send data to them. Period. Stop worrying about the low-level networking like NAT traversal, rendezvous, etc.

Start developing

No server-side code needed!

Ship your web app using static web servers and CDNs, or use someone else's for free like GitHub Pages. Package your code in a node.js package or browser extension. It's just JavaScript.

Plug-n-play networks

Want to use a different social network? No problem! Our plug-and-play API providers let you swap out the implementations behind the freedom.js interface, allowing your application to support multiple social networks, storage systems, and network stacks without changing your application code. Bootstrap your application with Google+, Facebook, or our global freedom.js social network today. (more coming soon). Or write your own!

Safely link code

When you host your freedom.js modules on the Web, any developer can link their app to your code, forming dynamic dependencies that are strongly isolated at runtime.
