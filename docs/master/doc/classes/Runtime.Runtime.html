Runtime.Runtime Class

Defined in: src/port-runtime.js:148
Module: freedom

A Runtime, backing the 'core.runtime' API. The runtime object handles requests by local applications wanting to interact with the freedom.js runtime. Primarily, this is done by using 'createApp' to connect with a remote application.
