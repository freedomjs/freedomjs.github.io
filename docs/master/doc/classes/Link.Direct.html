Link.Direct Class

Uses
  • handleEvents
Extends Port
Defined in: src/link/direct.js:8
Module: freedom

A port providing message transport between two freedom contexts in the same namespace. Note that using a direct link does not provide the isolation that freedom.js encourages. To that end it should be limited to a method for testing and not used in production without some serious though about the implications of that decision.
