getInterface

()

Defined in src/port-proxy.js:75

Create a proxy.Interface associated with this proxy. An interface is returned, which is supplied with important control of the proxy via constructor arguments: (bound below in getInterfaceConstructor)

onMsg: function(binder) sets the function to call when messages for this interface arrive on the channel, emit: function(msg) allows this interface to emit messages, id: string is the Identifier for this interface.
